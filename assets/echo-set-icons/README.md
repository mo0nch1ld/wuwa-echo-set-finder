# Echo set icons

Сюда можно положить иконки сетов.

Рекомендуемый формат: `PNG`, размер `64x64` или `128x128`.

Используй имена файлов ниже (или замени одноимённые `.svg` плейсхолдеры):

- `celestial-light.svg`
- `flaming-clawprint.svg`
- `freezing-frost.svg`
- `lingering-tunes.svg`
- `moonlit-clouds.svg`
- `rejuvenating-glow.svg`
- `sierra-gale.svg`
- `sun-sinking-eclipse.svg`
- `tidebreaking-courage.svg`
- `trailblazing-star.svg`
- `void-thunder.svg`

Если хочешь использовать PNG, просто назови файлы аналогично:
например `celestial-light.png`, `void-thunder.png`.
